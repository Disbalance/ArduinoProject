ArduinoProject
==============
Use sensor reostat, DHT11 and DS1302. Read value in sensor and write to LCD.
Pins:
DHT11 - 2 - analog pin;
Reostat - 0 - analog pin;
button1 = 3 - digit pin;
button2 = 7 - digit pin;
button3 = 8 - digit pin;
ledPanel = 9 - digit pin;
DS1302 rtc(4, 5, 6) -  (RST, DAT, CLK);
